Python变量

变量是存放数据值的容器，与其他编程语言不同，Python 没有声明变量的命令，首次为其赋值时，才会创建变量。

变量名称

变量可以使用短名称（如 x 和 y）或更具描述性的名称（age、carname、total_volume）。

Python 变量命名规则：

  • 变量名必须以字母或下划线字符开头
  • 变量名称不能以数字开头
  • 变量名只能包含字母数字字符和下划线（A-z、0-9 和 _）
  • 变量名称区分大小写（age、Age 和 AGE 是三个不同的变量

实例
# 合法的变量名:
myvar = “John”
my_var = “John”
_my_var = “John”
myVar = “John”
MYVAR = “John”
myvar2 = “John”

# 非法变量名:
2myvar = "John"
my-var = "John"
my var = "John"

创建变量

实例
x = 5
y = “John”
print(x)
print(y)

全局变量

在函数外部创建的变量（如上述所有实例所示）称为全局变量。

全局变量可以被函数内部和外部的每个人使用。

要在函数内部创建全局变量，您可以使用global关键字。

Python 数据类型

  • 文本类型: str
  • 数值类型: int, float, complex
  • 序列类型: list, tuple, range
  • 映射类型: dict
  • 集合类型: set, frozenset
  • 布尔类型: bool
  • 二进制类型: bytes, bytearray, memoryview

使用 type() 函数获取任何对象的数据类型:

Python 类型转换

  • int() 用整数字面量、浮点字面量构造整数（通过对数进行下舍入），或者用表示完整数字的字符串字面量
  • float() 用整数字面量、浮点字面量，或字符串字面量构造浮点数（提供表示浮点数或整数的字符串）
  • str() 用各种数据类型构造字符串，包括字符串，整数字面量和浮点字面量

实例

x = int(1)   # x 将是 1
y = int(2.8) # y 将是 2
z = int("3") # z 将是 3

x = float(1) # x 将是 1.0
y = float(2.8)   # y 将是 2.8
z = float("3")   # z 将是 3.0
w = float("4.2") # w 将是 4.2

Python 字符串

通过使用变量名称后跟等号和字符串，可以把字符串赋值给变量:
实例

a = "Hello"
print(a)

您可以使用三个引号将多行字符串赋值给变量:
实例

a = """Lorem ipsum dolor sit amet,
consectetur adipiscing elit,
sed do eiusmod tempor incididunt
ut labore et dolore magna aliqua."""
print(a)

字符串是数组

a = "Hello, World!"
print(a[1])
#索引从0开始

裁切

b = "Hello, World!"
print(b[2:5])
# 切片包含左索引，不包含右索引

负的索引

b = "Hello, World!"
print(b[-5:-2])
#负索引从字符串末尾开始

如需获取字符串的长度，请使用 len() 函数。

字符串方法

# strip() 方法删除开头和结尾的空白字符：
a = " Hello, World! "
print(a.strip()) # returns "Hello, World!"

# lower() 返回小写的字符串：
a = "Hello, World!"
print(a.lower())

# upper() 方法返回大写的字符串：
a = "Hello, World!"
print(a.upper())

# replace() 用另一段字符串来替换字符串：
a = "Hello, World!"
print(a.replace("H", "J"))

# split() 方法在找到分隔符的实例时将字符串拆分为子字符串：
a = "Hello, World!"
print(a.split(",")) # 返回 ['Hello', ' World!']

Python 逻辑运算符

运算符 描述
and 如果两个语句都为真，则返回 True
or 如果其中一个语句为真，则返回 True。
not 反转结果，如果结果为 true，则返回 False

Python 身份运算符

运算符 描述
is 如果两个变量是同一个对象，则返回 true。
is not 如果两个变量不是同一个对象，则返回 true。

Python 成员运算符

运算符 描述
in 如果对象中存在具有指定值的序列，则返回 True。
not in 如果对象中不存在具有指定值的序列，则返回 True。

Python 位运算符

位运算符用于比较（二进制）数字：

运算符 实例 等同于
& AND 如果两个位均为 1，则将每个位设为 1。
OR
^ XOR 如果两个位中只有一位为 1，则将每个位设为 1。
~ NOT 反转所有位。
<< Zero fill left shift 通过从右侧推入零来向左移动，推掉最左边的位。
>> Signed right shift 通过从左侧推入最左边的位的副本向右移动，推掉最右边的位。

Python 列表

列表是一个有序且可更改的集合，可以通过引用索引号来访问列表项。

** 实例 ** 

thislist = ["apple", "banana", "cherry"]
thislist.append("orange") #添加项目
thislist.remove("banana") #删除项目
thislist.pop()            #删除指定的索引（如果未指定索引，则删除最后一项）
del thislist[0]           #关键字删除指定的索引
thislist.clear()          #清空列表
mylist = thislist.copy()  #复制列表
thislist.extend(mylist)   #方法将 mylist 添加到 thislist 的末尾

Python 元组

元组（Tuple）

元组是有序且不可更改的集合，通过引用方括号内的索引号来访问元组项目。

实例

thistuple = ("apple", "banana", "cherry") #元组一旦创建，您就无法向其添加/删除项目。元组是不可改变的
tup1 = (50,) #元组中只包含一个元素时，需要在元素后面添加逗号 ,

Python 集合

集合（set）是一个无序的不重复元素序列。

集合中的元素不会重复，并且可以进行交集、并集、差集等常见的集合操作。

可以使用大括号 { } 创建集合，元素之间用逗号 , 分隔， 或者也可以使用 set() 函数创建集合。
